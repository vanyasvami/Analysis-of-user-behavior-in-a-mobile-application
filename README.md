# Анализ пользовательского поведения в мобильном приложении
1. Сферы деятельности: Бизнес,Интернет-сервисы,Стартапы
2. Направление деятельности: Маркетинг-аналитик,Продуктовый аналитик
3. Задачи проекта: На основе данных использования мобильного приложения для продажи продуктов питания проанализировать воронку продаж, а также оценить результаты A/A/B-тестирования 
4. Навыки и инструменты: A/B-тестирование,Matplotlib,Pandas,Plotly,Python,Seaborn,визуализация данных,проверка статистических гипотез,продуктовые метрики,событийная аналитика
5. Описание проекта: В данном проекте мной были изучены принципы событийной аналитики. Я построил
воронку продаж, исследовал путь пользователей до покупки. Проанализировал
результаты A/B-теста введения новых шрифтов. Сравнил 2 контрольных группы между
собой, убедился в правильном разделении трафика, а затем сравнил с тестовой группой
Выявлено, что новый шрифт значительно не повлияет на поведение пользователей.
